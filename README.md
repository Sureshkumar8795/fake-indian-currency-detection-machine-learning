# 💰 Fake Indian Currency Detection Using Machine Learning

A Machine Learning project that detects whether an Indian currency note is **Real** or **Fake** using image processing and feature extraction techniques.

The system preprocesses the uploaded currency image, extracts meaningful features, and classifies the note using a machine learning model to improve counterfeit currency detection.

---

## 📖 Project Overview

Counterfeit currency is a serious challenge that affects the economy. This project provides an automated solution for identifying fake Indian currency notes using image processing and machine learning techniques.

The application extracts important visual features from currency images and predicts whether the note is genuine or counterfeit.
